# huilin-world-forge
慧凌世界地形系统框架

## 项目介绍
本项目是一套标准化AI世界地形内容生产架构，
采用分层固定结构：核心协议、地形蓝图、渲染引擎、声景生成、世界装配、稳定性扫描。

统一规则锁定、风格一致、可批量生成虚拟场景与剧情内容。

## 目录结构
0_CORE_PROTOCOL      核心世界协议层
1_TERRAIN_BLUEPRINTS 地形场景蓝图层
2_RENDER_ENGINE      AI渲染引擎层
3_SOUND_GENERATOR    声景音频生成层
4_WORLD_BUILDER      世界装配导出层
5_SCANNER            规则稳定性校验层
